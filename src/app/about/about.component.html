A long time ago in a galaxy far far away...

The proposed of this project was created an application with information about the Star Wars Universe and to check if the developer has the right skills in Angular 2++ to have a position in a new team. This project was created using Angular version 6.1.10 with CLI version 6.2.6 and I'm using the SWAPI (Star Wars API) to consume the Star Wars data. Also, I'm using Bootstrap version 3.3.7 and icons from Font Awesome to help with the CSS styles.

New Features

I created a service to show in the every page bottom a quote from some character. This service will get this information from an array in a random way, so the footer will be dynamic.
